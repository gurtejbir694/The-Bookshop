# ReadNook – a personal reading corner online

## E-commerce Bookshop Project

- Online bookstore website for book lovers.
- User features: signup/login, browse books by category.
- View book summaries, author details, and prices from sites like Amazon.
- Explore other works by the same author.
- Free section: download or read PDF of selected books.
- Easy way to discover and buy books offline or online.
- Contact form available for user queries
